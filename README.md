# Scratch Game AI Bot

**Automatically generates Scratch game ideas, step-by-step instructions, and Scratch block examples.**  

This bot is designed to help you **plan and design Scratch games** automatically. It provides full game concepts, player/enemy details, scoring, levels, clone management, power-ups, and extra features — all in one script.

---

## Features

- Fully automatic, requires no user input  
- Chooses a random game type (platformer, fighting, clicker, horror, dodge, adventure, maze, runner, puzzle, survival)  
- Searches the web for Scratch game inspiration  
- Generates random **player characters, enemies, and goals**  
- Provides **step-by-step instructions** (50+ detailed steps)  
- Suggests **Scratch blocks for movement, enemies, clones, collisions, powerups, game over, and win conditions**  
- Recommends **extra features** like boss battles, collectibles, timer challenges, unlockable skins, and more  
- Designed for **personal or educational use only**
